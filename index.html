GIT AND GITHUB
git init
To initialise git into a particular file.
git remote add origin
To add connect the github repository with our git.
git status
To see if our particular file is staged or not .
git add
To stage the particular file to be added in our local machine.
git remote -v
to verify the remote repo working.
git commit -m
To describe our change(in our local machine).
git push
To reflect our code on the github repository.
git pull
to fetch the code from github repository.
git log
To see the record of our made changes.
git config --list
To see basic system info.
git clone url
To clone a particular repository in our local system.
git branch
To check the branch on which we are
.git checkout -b name
to make new branch
git branch -d name
to delete a branch
git branch checkout name
to shift to a new branch
